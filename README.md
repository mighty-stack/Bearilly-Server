# Bearily Backend

## Production checklist
- Set the required environment variables for MongoDB, JWT, and email services.
- Ensure the uploads directory exists and is writable.
- Run tests with `npm test`.
- Start the API with `npm start`.

## Deployment notes
- Deploy the Node.js backend to Vercel, Render, Railway, or a similar host.
- Use Supabase or Firebase Storage for uploaded files if you need persistent public storage.
- Configure CORS and environment variables in the hosting platform.
